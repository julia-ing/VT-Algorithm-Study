    for i in range(len(nums)):

## 2. longest increasing subsequence

### ğŸ”® My solution

- ì²«ë²ˆì§¸ë¡œ ì‹œë„í•œ ì½”ë“œëŠ” commoní•œ dynamic programming ë°©ì‹
- ì‹œê°„ ë³µì¡ë„: O(N^2)
  - ì˜ˆì‹œ:
    - nums = [10,9,2,5,3,7,101,18]
    - dp 
      - i = 0: [1,1,1,1,1,1,1,1]
      - i = 1: [1,1,1,1,1,1,1,1]
      - i = 2: [1,1,1,1,1,1,1,1]
      - i = 3: [1,1,1,2,1,1,1,1]
      - i = 4: [1,1,1,2,2,1,1,1]
      - i = 5: [1,1,1,2,2,3,1,1]
      - i = 6: [1,1,1,2,2,3,4,1]
      - i = 7: [1,1,1,2,2,3,4,4]
      - result : max 4

```python
def lengthOfLIS(self, nums):
    dp = [1] * len(nums)
    for i in range(len(nums)):
        for j in range(i):
            if nums[j] < nums[i]:
                dp[i] = max(dp[i], dp[j] + 1)
    return max(dp)
```
